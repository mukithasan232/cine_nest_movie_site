# 🍿 CineNest Movie Engine

A premium, high-end Movie Information & Review platform built with **Next.js 14+ (App Router)**, **Tailwind CSS**, and **Framer Motion**.

## ✨ Features
- **Cinematic Dark Design**: A visually stunning UI built with `zinc-950` and high-fidelity accents.
- **TMDB Live Data**: Real-time integration for Trending and Upcoming movies.
- **ISR Architecture**: Optimized with a 3600s revalidation cycle for peak SEO and performance.
- **High-Performance Logic**: Optimized with `next/image` and server-side data fetching.
- **Premium Glassmorphism**: High-fidelity detail containers and interactive cast/related movie grids.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mukithasan232/cine_nest_movie_site.git
   cd cine_nest_movie_site
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env.local` file and add your TMDB API Key:
   ```env
   NEXT_PUBLIC_TMDB_API_KEY=your_api_key_here
   ```

4. **Run in development**:
   ```bash
   npm run dev
   ```

## 🌍 Vercel Deployment

1. Push your code to GitHub (Done).
2. Connect your repository to [Vercel](https://vercel.com/new).
3. **Important**: Add `NEXT_PUBLIC_TMDB_API_KEY` to your Vercel Project Environment Variables.
4. Vercel will automatically build and deploy with ISR support enabled!
